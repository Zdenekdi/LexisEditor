# LexisEditor: AI-Native Legal Word Processor

LexisEditor: AI-native náhrada MS Word pro advokáty. Stabilní právnické číslování, automatická anonymizace a hluboká integrace lokální AI (Ollama). Práce se strukturovanými daty pro přesnou rešerši a stylizaci dokumentů. Soukromí na prvním místě, 100% offline běh. Profesionální legal-tech pro Windows, Mac i Linux.
